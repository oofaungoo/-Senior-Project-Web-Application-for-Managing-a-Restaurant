## How to Run the Project
1. Clone the Repository
   ```
   https://github.com/oofaungoo/-Senior-Project-Web-Application-for-Managing-a-Restaurant
   ```
2. Install Dependencies

   Front-end
   ```
   cd FoodMood
   npm install
   ```
   Backend
   ```
   cd backend
   npm install
   ```
4. Run the Project

   Front-end
   ```
   cd FoodMood
   npm start
   ```
   Backend
   ```
   cd backend
   npm start
   ```
   The app will be available at: ```http://localhost:3000```
